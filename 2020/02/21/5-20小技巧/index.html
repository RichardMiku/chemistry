准备工作

工作环境

系统环境

  • Microsoft Windows 7 或更高版本

软件环境

  • Visual Studio 2012 或更高版本
  • .Net Framework 4.5
  • 酷Q Air/Pro
    • Air：选择此版本您可以做一些文字小游戏和对方娱乐
    • Pro：Pro支持图片、语音和卡片功能，因此选择此版本您将做的更为浪漫

开启开发者模式

  • 开启开发者模式教程来源于酷Q文库

步骤：悬浮窗右键 > 应用 > 应用管理，左键点击应用窗口右下角版本号5次，在弹出的窗口中开启开发模式并重启。
开启开发模式后，酷Q将新增以下功能，帮助您进行开发：

  • 错误信息将详细显示。
  • 支持载入、打包未打包应用。（开启开发模式后才能载入开发中的应用）
  • 在应用窗口，按住shift并单击“重启酷Q”，可以快速重载应用（而无需重启）。
